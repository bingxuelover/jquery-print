# 功能点

打印功能的jquery实现

# 项目运行

## 初始化

- yarn

## 启动项目

- yarn start

# 技术点

## jquery.PrintArea.js

- index.html

### 功能

- IE 浏览器页面设置网页打印的页眉页脚为空
- iframe 形式弹出 和 popup 弹出

### 参数

- mode 弹出模式，默认iframe， 可以设置为 popup
- popWd 弹窗宽度

## jquery.print.js

- jqprint.html

### 功能

- IE 浏览器默认设置网页打印的页眉页脚为空
- iframe 形式弹出

### 参数

- importCSS 设置弹出打印区域的样式
